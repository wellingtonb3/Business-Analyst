
# 🚀 Corporate Strategy AI: Verified Strategic Advisor

**AI Engineering | Grounding & Verification | Oracle Cloud Infrastructure**

## 📌 Project Overview

The **Corporate Strategy AI** is a high-performance, cloud-native application designed to bridge the gap between generic AI outputs and actionable business intelligence. Unlike standard LLMs that may hallucinate, this tool implements **Google Search Grounding** to verify the real-world existence and current market situation of organizations before generating a report.

Built as part of my journey as an **Oracle ACE Apprentice**, this project demonstrates the integration of advanced LLMs with enterprise-grade cloud infrastructure to solve real-world management challenges in the Brazilian market.

-----

## 🏗 System Architecture

The application is hosted on a professional cloud stack designed for 24/7 availability and low latency:

### 🖥️ Cloud Infrastructure (The Engine)

  * **Hosting:** **Oracle Cloud Infrastructure (OCI)** Virtual Machine (VM).
  * **Environment:** Python-based Virtual Environment (`venv`) for isolated dependency management.
  * **Security:** Enterprise-grade VCN (Virtual Cloud Network) security rules to manage external traffic on port **8501**.

### 🧠 AI Logic (The Intelligence)

  * **Model:** **Gemini 2.5 Flash** (optimized for speed and complex reasoning).
  * **Grounding:** Integrated **Google Search Tool** to perform live web verification.
  * **Logic:** The system utilizes a "Verification Gate"—if a company cannot be verified through live data, the model refuses to generate a fictional report, ensuring data integrity.

## 🚀 Engineering Highlights

  * **Google Search Grounding:** Ensures 0% hallucination for non-existent companies by checking live web indexes.
  * **Streamlit Framework:** Provides a clean, reactive frontend for business users to interact with complex AI models.
  * **Brazil Market Focus:** Specialized system instructions to prioritize Brazilian economic context, logistics challenges (e.g., fuel costs), and local management culture.
  * **Scalability:** Hosted on OCI, allowing the application to scale from a single user to an enterprise-level tool with minimal configuration changes.
